# 📋 Coding Test Study (Algorithm)

백준(Baekjoon Online Judge) 및 프로그래머스 문제 풀이 기록 저장소입니다.  
[백준허브(BojHub)](https://github.com/BaekjoonHub/BaekjoonHub)를 이용해 자동으로 풀이 소스코드가 업로드됩니다.

---

## 🚀 학습 목표
- 바킹독의 실전 알고리즘 강의 커리큘럼 따라잡기
- 파이썬(Python)을 활용한 효율적인 알고리즘 구현 능력 배양

## 🛠️ 환경 설정
- **Language**: Python 3.x
- **Platform**: Baekjoon, Programmers
- **Tool**: Visual Studio Code, BaekjoonHub (Chrome Extension)

## 📌 문제 풀이 현황

| 구분 | 상태 |
| :-- | :-- |
| 기초 코드 작성 요령 | ✅ 완료 |
| 배열 / 연결 리스트 | 🏃 진행 중 |
| 스택 / 큐 / 덱 | ⏳ 대기 |
| BFS / DFS | ⏳ 대기 |
| 정렬 / 다이나믹 프로그래밍 | ⏳ 대기 |

---

## 💡 유의 사항 (Pythonic Coding)
https://github.com/hanXen/basic-algo-lecture-python
위 레포에 작성된 파이썬 정답 코드들을 참고하여 진행하고 해당 코드 스타일도 따라갈 예정
- **빠른 입출력**: `sys.stdin.readline` 활용
- **구조화**: `main()` 함수와 `solve()` 로직의 분리
- **타입 힌트**: `def solve(n: int) -> None:` 와 같이 가독성 증대

- `프로그래머스/`: 프로그래머스 문제 풀이 소스코드
